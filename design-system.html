02 — Typography

Type system

A high-contrast display serif carries headlines and voice-led statements; a geometric sans handles body copy, UI and labels.

Aa Denton

Denton Bold — Display

Licensed font (Colophon Foundry). Shown here with Fraunces as an open-source stand-in with similar high-contrast, display-serif character. Used at 700 weight for all headings and hero statements.

700 Bold

Aa Gotham

Gotham A / B — Body & UI

Licensed font (Hoefler&Co), loaded as a split "A/B" variable pairing. Shown here with Montserrat as a geometric-sans stand-in. Used from 200–700 weight across body copy, buttons and labels.
